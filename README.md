# Hospital Patient Priority Handling Application

## Project Description

The **Hospital Patient Priority Handling Application** is a web-based system designed to manage patient registration and serve patients based on their medical priority.
Patients are assigned different priority levels such as **Critical, High, Moderate, Mild, and Low**, and the system ensures that patients with higher priority are served first.

This project demonstrates the use of **Data Structures (Priority Queue concept)** in a real-world hospital management scenario.

-----------------------------------------------------------------------------------------------------------------------------

## Features

* Patient registration form
* Assign priority levels to patients
* Display registered patients in a table
* Automatically sort patients based on priority
* Serve patients according to priority level
* Simple and responsive user interface

----------------------------------------------------------------------------------------------------------------------------

## Technologies Used

* **HTML** – Structure of the web page 
* **CSS** – Styling and layout of the application 
* **JavaScript** – Logic for patient handling and priority sorting 

----------------------------------------------------------------------------------------------------------------------------

## Data Structure Used

The project uses the **Priority Queue concept** to ensure that patients with higher priority are served before others.

Priority order implemented in the system:

1. Critical
2. High
3. Moderate
4. Mild
5. Low

Patients are automatically sorted based on these priority levels.

---------------------------------------------------------------------------------------------------------------------------

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open the `index.html` file in any web browser
4. Add patient details in the form
5. Click **Add Patient**
6. Click **Serve Next Patient** to serve the highest priority patient
